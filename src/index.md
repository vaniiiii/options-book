# Options Mastery — From Zero to Black-Scholes

A 15-module course that takes you from "what is an option?" to understanding the Black-Scholes pricing model — with real examples, Python code, and no hand-waving.

## Who This Is For

You know what a stock is, what buying and selling means, and that prices change. That's it. No finance degree required.

## How to Use This Book

Each module builds on the previous one. Work through them in order. Every module includes:

- **Worked examples** with real ticker symbols and realistic numbers
- **Python code** you can run yourself
- **Quizzes** to test your understanding (answers hidden behind spoiler tags)
- **Common mistakes** that trip up beginners
- **A glossary** of every new term introduced

Start with [Module 1: What Options Are](./what-options-are.md).
